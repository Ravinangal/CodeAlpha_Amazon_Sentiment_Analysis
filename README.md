# Amazon Review Sentiment Analysis

## Project Overview
This project analyzes Amazon product reviews using Python.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Sentiment Analysis

## Technologies Used
- Python
- Pandas
- Matplotlib
- TextBlob

## Key Insights
- Most ratings were polarized between 1-star and 5-star reviews.
- Positive reviews slightly exceeded negative reviews.
- Customer sentiment showed strong emotional responses.
